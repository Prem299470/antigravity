# Portfolio

This folder contains a static cybersecurity portfolio website for Prem Ranjith Varma K.

## Files

```text
portfolio/
|-- assets/
|   `-- Prem_Ranjith_Varma_Resume.pdf
|-- index.html
|-- script.js
|-- styles.css
`-- README.md
```

## Purpose

The portfolio presents:

- cybersecurity profile and career focus
- role-aligned skills in penetration testing, threat detection, and automation
- selected security projects with measurable outcomes
- education, certifications, achievements, and contact links

## Tech Stack

- `HTML` for structure and content
- `CSS` for layout, theme styling, and responsive presentation
- `JavaScript` for light theme persistence and interaction

## Local Preview

Open `index.html` directly in a browser to review the site locally.

## Deployment

Because the site is fully static, it can be deployed to GitHub Pages, Netlify, Vercel static hosting, or any standard web server without a build step.
